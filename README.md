## Running the app

````bash
# Download git

sudo yum install -y git

# Set execute permission

chmod +x run.sh

sh run.sh

source ~/.bashrc

# Docker mongoDB

docker-compose up -d

# Start

npm run start


# Endpoints

1. Seed database

```
http://localhost:4000/api/v2/seed

```

2. Get all pokemons

```
http://localhost:4000/api/v2/pokemo

```

3. Get pokemon by id

```
http://localhost:4000/api/v2/pokemon

```
````
