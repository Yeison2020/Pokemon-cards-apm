## Running the app

```bash
# Download git

sudo yum install -y git

# Set execute permission

chmod +x run.sh

sh run.sh

# Start

# npm install

npm run start

## Stack

- Mongo DB
- TablePlus UI to view Database details
- NestJS v9.0.0
- Express v4.17.13
- Nodejs v18.15.11
```
